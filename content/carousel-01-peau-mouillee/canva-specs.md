# Canva Design Specifications — Carousel 1

## Format
- **Platform:** Instagram Carousel
- **Dimensions:** 1080 x 1350 px (4:5 ratio — maximum feed real estate)
- **Slides:** 7
- **File format:** PNG export, high quality

## Brand Kit

### Colors
| Token   | Hex       | Usage                                    |
|---------|-----------|------------------------------------------|
| Navy    | `#1a2744` | Primary text, headlines                  |
| Cream   | `#faf8f5` | Backgrounds, text on dark photos         |
| White   | `#ffffff` | Cards, overlays                          |
| Peach   | `#f5c4aa` | Accents, highlights                      |
| Peach-L | `#fdeee5` | Soft backgrounds, gradient stops         |
| Rose    | `#8a6565` | Secondary text, subtle accents           |
| Gold    | `#f5a623` | Icons, badges, premium feel              |
| Muted   | `#8a8a8a` | Captions, secondary info                 |

### Typography
| Element          | Font                          | Weight    | Size (px) |
|------------------|-------------------------------|-----------|-----------|
| Slide headline   | Cormorant Garamond            | Semi-Bold | 64-80     |
| Sub-headline     | Cormorant Garamond            | Italic    | 36-44     |
| Body text        | DM Sans                      | Regular   | 28-32     |
| Labels/tags      | DM Sans                      | Medium    | 24-28     |
| CTA              | DM Sans                      | Medium    | 32-36     |
| Korean accents   | System (Noto Sans KR)        | Regular   | 20-24     |

### Logo
- "Sonagi" in Cormorant Garamond, Semi-Bold
- Small, bottom-right corner of every slide (subtle watermark)
- Color: Navy (#1a2744) on light images, Cream (#faf8f5) on dark images
- Opacity: 60%

## Canva Template Workflow

1. **Create template:** 1080x1350px, background #faf8f5
2. **Add brand fonts:** Cormorant Garamond + DM Sans (both on Google Fonts, available in Canva)
3. **Set brand colors:** Add all hex values above to your Canva Brand Kit
4. **For each slide:**
   - Place Nano Banana generated image as background/main element
   - Add text overlay elements per layout above
   - Apply semi-transparent bars where specified
   - Place Sonagi logo watermark bottom-right at 60% opacity
5. **Export:** PNG, high quality, all slides
