## HOOK FORMULAS THAT WORK ON SKINCARE TIKTOK

| Formula | Example |
|---------|---------|
| Vulnerable confession | "My skin was DESTROYED" |
| Stranger compliment | "A girl stopped me to ask THIS" |
| Celebrity reference | "Jungkook made me try it" |
| Reaction/shock | "Korean stars use THIS for their pores" |
| Challenge/provocative | "Don't even try this" |
| Before/after tease | "One product changed EVERYTHING" |
| Clinic authority | "I got my skin analyzed in Korea" |
| Counter-intuitive | "The WORST beauty trend from Korea" |

---

## DIALOGUE RULES

- **30-40 words per 15-second clip** (in target language)
- Start mid-thought — no "Hey guys" or "So today I'm gonna..."
- Use short punchy sentences, not complex grammar
- Include ONE product name mention per clip (natural, not forced)
- End Clip C with a warm personal sign-off, not a sales CTA
- The tone is always: friend sharing a secret, never: salesperson

---

## SCRIPT SERIES PLANNING

For a 5-video series, vary the hook format:

| Video | Hook Type | Emotional Arc |
|-------|-----------|---------------|
| 1 | Macro/visual hook | Curiosity → satisfaction |
| 2 | Celebrity reaction | Shock → proof |
| 3 | Authority (clinic/expert) | Trust → conviction |
| 4 | Story (stranger/compliment) | Relatability → aspiration |
| 5 | Vulnerability (before/after) | Empathy → care |

Each video features a different product but the same character, same bathroom, same energy.
